WHAT THIS IS
Station 05 of the ShortFactory SF Studio pipeline. Multi-channel Publisher.
Raw content in → formatted, optimised copy for every channel out.
Used after Station 04 Valuator confirms the content scores high enough to publish.

THE 5-STAGE PIPELINE
01 Content — paste source material, set tone + audience
02 Channels — pick which platforms to publish to (up to 8)
03 Generate — Grok writes optimised copy for each channel simultaneously
04 Schedule — pick best days/times to post (display only — no auto-post yet)
05 Publish — review all outputs, copy to clipboard, post manually

SUPPORTED CHANNELS
X/Twitter (280 chars) · LinkedIn (long-form) · Reddit (title + body)
Hacker News (title only, max 80 chars) · Email (subject + body)
Press Release (formal, inverted pyramid) · YouTube (title + desc + tags)
Short Script (60–90s spoken, hook-first)

HOW IT WORKS
1. User pastes content + selects tone + audience in Stage 01
2. Selects channels in Stage 02
3. JS builds one prompt covering all selected channels → fires to Grok
4. Grok returns JSON keyed by channel name
5. Each output renders in a copy block with one-click clipboard copy
6. Copy All dumps every channel's output to clipboard in labelled blocks

API
Grok proxy: https://api.shortfactory.shop/grok/chat (POST)
Model: grok-4-latest · Temperature: 0.5 (slightly warmer for creative writing)
XAI key stored as Cloudflare secret XAI_KEY — never in source
