# PRODUCT DESIGN BRIEF

## Project: Multi-Eye Scan → 3D Concept Engine (ShortFactory)

### 🎯 Core Idea

Users scan real-world objects with their phone. The system converts scans into a **3D visual object + concept signature**, rewarding users instantly with **ShortFactory (SFT) credits**.

The experience should feel like:

> “I just discovered and captured something — and got rewarded instantly.”

---

# 🧠 User Loop (CRITICAL)

### 1. Capture

* User points camera at object
* Guided UI encourages moving around it (multi-angle “many eyes”)

### 2. Build (Instant Feedback)

* Object begins forming in **live 3D (wireframe → solid)**
* User sees shape emerging in real time

### 3. Solve Moment ⚡ (Dopamine Peak)

* System recognizes or approximates object
* Displays:

  * “This looks like: SHOE (82% match)”
* OR:

  * “New object discovered”

### 4. Reward 💰

* Instant SFT credit drop
* Visual + audio feedback (VERY important)

### 5. Store

* Object added to:

  * personal collection
  * global library

---

# 🎨 Visual Direction

## Core Feeling:

* “Futuristic scanner meets playful discovery”
* Think:

  * holographic
  * glowing wireframes
  * evolving geometry

---

## 3D Object States

### 1. Scan Phase

* particles / dots forming structure
* incomplete mesh
* noisy → stabilizing

### 2. Solve Phase

* snap into clean shape
* glow pulse
* label appears

### 3. Reward Phase

* credits burst outward
* satisfying pop / ripple
