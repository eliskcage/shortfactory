❌ Can't Do Yet

Priority order — top = most painful gap. Fix label = effort required.
1
Real-time voice conversations with ALIVE
PersonaPlex 7B gives full-duplex speech-to-speech at 70ms. ALIVE can type but not talk back. This is the single biggest missing superpower — the creature needs a mouth.
Big build — RTX 4090 + PersonaPlex integration
2
Auto-send outreach emails
The pipeline writes a perfect personalised email but doesn't pull the trigger. Resend.com API key needed — sign up free, add RESEND_API_KEY to CF Workers dashboard → ai-outreach → Settings → Variables. That's it.
5 min fix — resend.com free tier + one CF dashboard step
3
Hourly autonomous pipeline cron
ai-outreach has a scheduled() handler ready. Blocked because CF requires a workers.dev subdomain before cron triggers activate. One visit to CF dashboard Workers page auto-creates it.
2 min fix — visit CF dashboard once to register workers.dev
4
Auto-book follow-up Google Calendar events
Code is written. Needs a Google Calendar OAuth token (GCAL_TOKEN) in the ai-outreach Worker. Requires Google Cloud Console OAuth setup.
Medium — Google OAuth app + token rotation
5
Reliable browser screenshots of live SF pages
ai-browser works but times out on pages with continuous animation/polling (game, ALIVE, screensaver). networkidle0 strategy waits forever. Being fixed now.
Deploying fix now — switching to domcontentloaded
6
Token economy / smart contracts
SFT was killed. The full vision (contribution credits → token → smart contract → payout) has no on-chain component yet. Rank tiers exist in UI only — nothing mints or pays out.
Big build — ICP/Filecoin + smart contract + payout rails
7
Play Store app published
Keystore exists (ideafactory-new.jks), APK buildable. Not submitted. Needs Google Play Console account, store listing, screenshots, privacy policy URL.
Medium — store listing + review process (7–14 days)
8
stinkindigger.* domains live
Domains registered (.info/.com/.store/.co.uk), server running at 185.230.216.235, but DNS A records not yet pointed. No content deployed to those domains.
1 hr — DNS update + landing page deploy
9
Distributed compute payouts
Screensaver harvests GPU/CPU. Contribution is tracked in UI with rank tiers but no actual crypto payout to contributors. Battery economy is score-only.
Big build — crypto wallet integration + payout triggers
10
Multi-agent Worker swarm
All CF Workers are independent and siloed. No coordination, no shared state beyond D1. No agent-to-agent communication, job queuing, or emergent task routing.
Medium-large — CF Queues + Durable Objects for coordination
11
Auto-pin content to IPFS from pipelines
IPFS Vault page exists. Generated images, videos, and game states are not automatically pinned to Filecoin/Pinata. No persistence beyond server storage.
Medium — Pinata API wired to image/video generation events
12
YouTube auto-upload from Imaginator
Google OAuth and Drive integration exists. The final step — auto-upload assembled Short to YouTube channel — is not wired up.
Medium — YouTube Data API v3 upload call
